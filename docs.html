Create Test Schema

All of your tests must be in a schema with a name ending with _test.

CREATE SCHEMA myapp_test;

Example Test

This is an example test:

CREATE OR REPLACE FUNCTION myapp_test.test_alpha() RETURNS void
    LANGUAGE plpgsql
    VOLATILE
    SECURITY INVOKER
    AS $_$
-- module: example
-- submodule: basic
-- plan: 3
DECLARE
    _scores   dis.score[];
BEGIN
    _scores := ARRAY[
        dis.assert(TRUE, 'This is a test'),
        dis.assert(FALSE, 'Oops this will fail'),
        dis.assert(TRUE, 'This also works')
    ];
    PERFORM dis.tally(_scores);
END;
$_$;

in _test schema

prefix name with test_

comment with module, submodule, plan

collect array of dis.score types which are passed to dis.tally which throws
